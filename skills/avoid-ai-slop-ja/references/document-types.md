# Document types — what to adopt vs. exclude

The anti-slop techniques are universal in spirit but **not uniform in application**. "Edge" (毒・皮肉・自虐・誤字・カジュアル一人称・中間温度) suits personal writing and is wrong for formal docs. Always adopt: 具体性・削減・ヘッジ削減・false-agency 回避・構文チェック・リズムのムラ. The table sets the *bluntness/voice budget*.

| 文書種別 | 採用する | 除外/抑制する |
|----------|----------|----------------|
| 企業ブリーフィング | 具体性・削減・ヘッジ削減・構文チェック・控えめなムラ | 毒・皮肉・自虐・意図的な誤字・カジュアル一人称 |
| 法務・医療・金融・セキュリティ | 正確性・追跡可能性・根拠付き断定 | ムラより整合性優先。皮肉・誤字は禁止 |
| 技術ドキュメント | 主体性・具体性・手順の明確さ | 過度なリズム崩し・洒落・口語 |
| 採用広報・ブログ・エッセイ | ムラ・温度差・軽い皮肉 | わざとの誤字運用は避ける |
| SNS・個人名義の短文 | 毒・自虐・偏り（少量） | 人格攻撃・根拠なし断定 |
| 経営者メッセージ | 中間温度・一人称 | 過剰なくだけ方（信用を落とす） |
| 社内メモ | 粗さ許容 | 結論・依頼・期限・責任者は曖昧にしない |

## 原則

- **立場を取る**はどの種別でも必須。ただし「根拠を置いたうえで判断を曖昧にしない」であって、無根拠な断言や煽りではない。
- 不確実性が本物なら、消すのでなく根拠とともに残す（儀式的ヘッジとは別）。
- formal 文書で「人間らしさ」を出すのは、毒や誤字ではなく **具体・固有名・実データ・正直な留保** で行う。
- casual 文書（ブログ・SNS・エッセイ）で「人間らしさ」を出すのは、**書き手の不在を埋める**こと。casual のときはこの「書き手の復元」を最優先の処方にする。
  - 入れる: 自分の実体験での失敗・迷い・偏った実感・言い切り。中間温度・一人称・軽い毒も材料。
  - 入れない: 実体験がないのに**捏造した自分語り**（別種の嘘くささが出る。具体は実在の事実から取る）。過剰な毒・わざとの誤字。
